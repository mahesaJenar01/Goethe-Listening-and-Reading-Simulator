App to practice Goethe Listening and Reading Exam.

Use each of the system prompt to instantly generate listening of reading exam. The transcripts generated by the AI should then be converted into speech and then merge them.

Add static and add audios directory in the backend directory. Create new folders each names teilNUMBER (teil1, teil2). In each part there should be audios name listening-part-NUMBER-instanceNUMBER.mp3.
